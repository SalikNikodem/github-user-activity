# github-user-activity
Simple command line interface to fetch the recent activity of a GitHub user and display it in the terminal.

Listing recent github user activity
Filtering by event type

Installation
git clone [https://github.com/SalikNikodem/github-user-activity.git](https://github.com/SalikNikodem/github-user-activity.git)
or
gh repo clone SalikNikodem/github-user-activity

Use
cd github-user-activity

python user_activity.py [user]

to filter by specific events use

python user_activity.py [user] [event]
